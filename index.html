Premium Calculation

Formula:
Premium = (P2P_Rate - Theoretical_Rate) / Theoretical_Rate × 100
Where:
• P2P_Rate = Binance P2P market rate
• Theoretical_Rate = Crypto_Price × Exchange_Rate

Sample Implementation

// Premium calculation example
const btcPriceUSD = 43250;
const usdToVnd = 24500;
const binanceP2PRate = 1065000000; // VND per BTC

const theoreticalRate = btcPriceUSD * usdToVnd; // 1,059,625,000
const premium = ((binanceP2PRate - theoreticalRate) / theoreticalRate) * 100;
// Result: 0.51% premium in Vietnam
